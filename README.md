WinCenterClient
=========

WinCenter-Appliance虚拟化管理系统安装程序。


WinCenterClient使用C#编写，目前版本号为6.5。


说明
------
* 兼容XenServer7.0/WinServer6.5及以下所有版本
* 也可以作为虚拟机导入工具，将虚拟机导入到XenServer/WinServer主机

-------

* 优化性能
